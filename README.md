## 2. Projecció Professional

### 2.a. Especialització professional: Administrador de Sistemes Linux/Windows

L'especialització escollida és la d'**Administrador de Sistemes**, amb enfocament en entorns Linux i Windows Server. Aquest rol implica:

**Tasques principals:**
- Instal·lació, configuració i manteniment de servidors Linux (Debian, Ubuntu Server, CentOS) i Windows Server.
- Gestió d'Active Directory, polítiques de grup (GPO), DNS, DHCP i serveis de directori.
- Monitorització de sistemes i gestió de còpies de seguretat.
- Implementació de mesures de seguretat: firewalls, VPNs, hardening de sistemes.
- Gestió de virtualització amb Docker, VMware o VirtualBox.
- Automatització de tasques mitjançant scripts (Bash, PowerShell, Python).
- Suport tècnic de nivell 2/3 a usuaris i equips.

**Competències clau del rol:**
- Coneixement profund de sistemes operatius Linux i Windows.
- Capacitat de resolució d'incidències i diagnòstic de problemes.
- Coneixements de xarxes (TCP/IP, VLANs, routing).
- Mentalitat de seguretat i bones pràctiques en administració.
