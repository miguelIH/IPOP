Aquesta experiència m'ha aportat no només competències tècniques, sinó també maduresa professional i capacitat de treballar en contextos multiculturals i exigents.

Aquest document recull el meu currículum per competències, la meva projecció professional i el meu pla de promoció personal, amb l'objectiu de definir el meu camí com a futur tècnic especialitzat en sistemes.
